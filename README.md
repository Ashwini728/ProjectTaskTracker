# 📌 Planora – Smart Task & Team Management Platform

**Planora** is a full-stack web application designed to streamline project and task management for teams. With role-based dashboards, intuitive UI, AI-powered assistance, and real-time collaboration features, Planora makes productivity smarter and teamwork seamless.

---

## 🚀 Features

- 👥 **Role-Based Dashboard**  
  Separate views and controls for Admin, Project Manager, and Member.

- ✅ **Task Management**  
  Create, assign, track, and update tasks with priority and status indicators.

- 📊 **Task Progress Chart**  
  Visual summary of task status (To-Do, In-Progress, Done) per user.

- 📁 **Secure File Uploads**  
  Upload and view project-related documents per member.

- 📅 **Task Calendar**  
  Calendar view of deadlines and schedules.

- 📈 **Analytics Dashboard**  
  Project and task analytics with charts.

- 💬 **AI Chat Widget**  
  GenAI-based assistant for quick help and updates.

- 🔍 **Search Functionality**  
  Easily search through team members, tasks, and uploads.

---

## 🛠️ Tech Stack

**Frontend:**  
- React.js  
- Bootstrap 5  
- React-Router

**Backend:**  
- Node.js  
- Express.js  
- MongoDB (Mongoose)  
- Multer (for file uploads)

**AI:**  
- Custom Chat Widget powered by GenAI
